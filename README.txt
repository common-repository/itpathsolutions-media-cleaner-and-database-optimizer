== Description ==

The most powerful tool for clearing unused media from your website and optimizing your database to boost site performance

This plugin efficiently optimizes your media files and database, enhancing site performance to deliver a faster, smoother user experience.

Experience streamlined cleaning and optimization within a single powerful tool, elevating your website's speed and responsiveness to new heights.

==  Features ==
1. Media Cleaner 
      - View all media files utilized on your site.
      - Easily clear unused media not associated with any posts or pages.
      - Streamline media management by deleting unused files.
      - Detect media usage across your website effortlessly.
      - Save time by skipping the cleanup process for media files that are in use.
      
2.  Database Cleaner

      -  Inactive & unused Plugins cleaner
      -  Inactive & unused Themes cleaner
      -  Revision clean up
      -  Draft clean up
      -  Auto draft clean up
      -  Post meta clean up
      -  Comment meta clean up
      -  All comments clean up
      -  User meta clean up
      -  Term meta clean up
      -  Trashed post &  product clean ip
      -  Trashed post / product clean up
      -  Transients clean up
      -  Pingback & trackback clean up

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `itpathsolutions-media-cleaner-and-database-optimizer` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. You will find WP Optimizer in your WordPress admin screen.
